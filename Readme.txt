Тестовое задание:

Собрать окно Leaderboard используя предоставленную графику и реализовать логику добавления, удаления и редактирования игроков.

Окно Leaderboard
1.Окно Leaderboard содержит в себе таблицу с игроками и две кнопки Add и Remove
2.У игрока в таблице есть 3 поля, позиция в списке, никнейм и кол-во очков (позиция считается автоматически в зависимости от кол-ва очков)
3.Если кол-во игроков больше чем может поместиться в окне, должно быть доступно пролистывание игроков с помощью drag n drop
4.Кнопка Add открывает окно Add Player
5.Кнопка Remove удаляет выбранного игрока в списке, позиции в списке должно обновиться
6.Если кликнуть правой кнопкой мыши по записи игрока в списке должно открыться окно Edit Player

Окно Add Player
1.Окно Add Player содержит поля ввода ника и кол-во очков и две кнопки Add и крестик для закрытия окна
2.Если одно из двух полей не заполнено, кнопка Add не должна быть активна
3.Если пользователь заполнил все поля и нажал кнопку Add, окно Add Player должно закрыться, а в списке Leaderboard должна появиться новая запись с игроком и обновиться позиции в списке

Окно Edit Player
1.Тот же функционал что и в окне Add Player, но вместо кнопки Add, кнопка Edit
2.При нажатии на кнопку Edit окно Edit Player должно закрыться, а в списке Leaderboard измениться информация о выбранном игроке и обновиться позиции в списке
